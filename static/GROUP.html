There are 2 ways to manage membership of a group, either by dialing an activation code from the handset or by using the link below.

  • Dial 910
  • At the prompt enter the group number eg 1111
  • When prompted enter the 6 digit Activation Code shown above. eg 555555
  • Wait for confirmation that it has been successful
As an alternative you can dial the whole string in one go for example 9101111555555

Dialing 910 will simply add or remove that extension from the group, if adding it will always be added with no Ring Delay,
if using the Manage Group page you can choose to delay rining of certain extensions by 20sec to create a 2 stage group.
